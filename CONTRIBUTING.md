# Guia de Contribuição

Obrigado por considerar contribuir com o Repository Manager!

## Como Contribuir

1. **Reportar Bugs**
   - Use o Issues Tracker
   - Descreva o bug detalhadamente
   - Inclua passos para reproduzir

2. **Sugerir Features**
   - Abra uma issue com a tag "enhancement"
   - Explique por que a feature seria útil

3. **Pull Requests**
   - Fork o projeto
   - Crie uma branch para sua feature
   - Teste suas mudanças
   - Submeta o PR

## Padrões de Código

- Use indentação de 4 espaços
- Comente código complexo
- Siga o estilo Bash padrão
- Teste em diferentes sistemas
