# 🚀 Repository Manager - Biblioteca de Ferramentas

[![Version](https://img.shields.io/badge/version-1.0-blue.svg)](https://github.com/bhu01hackerspace/repo-manager)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![Bash](https://img.shields.io/badge/shell-bash-red.svg)](https://www.gnu.org/software/bash/)

Um gerenciador de repositórios poderoso e elegante, similar ao Tool-X, para organizar, baixar e executar todas as suas ferramentas de segurança e desenvolvimento em um só lugar.

## ✨ Features

- 📦 **Gerenciamento Centralizado** - Todos os seus repositórios em um único lugar
- 🔄 **Atualização Automática** - Mantenha suas ferramentas sempre atualizadas
- 🚀 **Download em Massa** - Baixe todos os repositórios de uma vez
- 🎯 **Execução Rápida** - Execute qualquer ferramenta diretamente do manager
- 📊 **Status em Tempo Real** - Veja quais repositórios precisam de atualização
- 💾 **Import/Export** - Compartilhe suas listas de repositórios
- 📝 **Log Completo** - Todas as operações são registradas
- 🎨 **Interface Colorida** - Visual intuitivo e amigável

## 📋 Pré-requisitos

- Linux/Unix/MacOS
- Bash 4.0+
- Git instalado
- Conexão com internet
